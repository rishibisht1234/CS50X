My CS50 Journey

I began my journey with Harvard’s CS50 by completing CS50P (Introduction to Programming with Python). This course strengthened my problem-solving skills and helped me write clean, structured programs using Python. It trained me to think in terms of logic, edge cases, and program flow rather than just syntax.

After completing CS50P, I moved on to CS50x (Introduction to Computer Science), where I am currently learning the deeper foundations of computing. CS50x has pushed me to understand how things work beneath the surface—memory, pointers, data representation, and algorithmic efficiency.

  • CS50P – What I Gained
    • Strong Python fundamentals and structured programming
    • File handling, data processing, and automation
    • Confidence in building small utilities and tools
  • CS50x – What I Am Learning
    • C programming, memory, and pointers
    • Algorithms and complexity analysis
    • SQL for databases
    • Web development with HTML, CSS, and JavaScript

Together, CS50P and CS50x are shaping how I think as a developer. One taught me how to program effectively, and the other is teaching me how computers actually work. This combination is building a strong and lasting foundation for everything I aim to create in the future.
