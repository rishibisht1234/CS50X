Projects

These projects reflect my interest in applying AI and programming to solve real-world problems. Each one focuses on a different aspect of software development—ML, APIs, UI, and system design.

  • Heart Risk Prediction App (Streamlit)
    • Built an ML-based web app to predict heart disease risk.
    • Deployed using Streamlit for easy public access.
    • Focus on data preprocessing, model inference, and clean UI.
  • AI-Powered Calculator
    • Calculator enhanced with a chatbot for natural-language problem solving.
    • Includes a canvas to draw mathematical problems.
    • Combines traditional computation with AI assistance.
  • Parkinson’s Disease Risk Predictor API
    • API that predicts risk using speech samples and handwriting input.
    • Processes audio and image-based features for inference.
    • Designed as a backend service for medical-style applications.
  • ChatMe – CLI Chatbot
    • Command-line chatbot with interactive conversations.
    • Includes built-in mini games.
    • Focus on logic, flow control, and user interaction in terminal.
